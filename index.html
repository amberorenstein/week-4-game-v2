Mathematical Bartender Game

You will be given a random target number at the start of the game

There are four drinks below. Each has a mystery value between 1 and 12. By clicking a drink you will add points to your number.

You win the game by matching your number to the target number.

You lose if your total score exceeds the target number.

Each time the game starts the values of the drinks and the target number will change
